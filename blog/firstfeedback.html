Eerste feedback

Mijn eerste feedback op de website was vooral positief, maar er zaten ook een paar verbeterpuntjes in, hier staan ze allemaal op een rijtje:

+ Het thema en de opmaak van de website zijn goed en de website is makkelijk ingedeeld.
+ De structuur van de code is goed en ook makkelijk te lezen.
+ Er zijn extra dingen toegevoegd zoals een dynamische navigatiebalk.

hier ga ik ga dieper in op de minpunten en hoe ik deze heb verbeterd:

- Er was veel herhaling te vinden op de site zelf. Dit bijvoorbeeld in de titels en navigatiebalk.
Dit is makkelijk opgelost door te herhalingen op te zoeken en deze weg te halen.

- Ook was er veel herhaling en onnodige code gevonden, dit vooral in de stylesheet.
Dit is ook makkelijk opgelost toen door de onnodige code weg te halen, te checken of alles nog werkt zoals daarvoor, en het dan op te slaan.

- De namen in de code waren erg onoverzichtelijk, denk aan genummerde classes.
Ook weer makkelijk opgelost door de namen naar iets overzichtelijkers te veranderen, zoals de naam van de plaatjes van 1-2-3 naar wat er op te zien is.

- Er miste een paar hoofd-onderdelen zoals een progressiebalk bij de tabel van mijn schoolopdrachten.
Hoewel deze progressiebalk er nu wel is, werkt deze nog niet uit zichzelf tenzij verandert in de code zelf,
dit omdat het nog niet nodig was om Javascript te gebruiken in de opdracht.

- De code mist opmerkingen
Hier heb ik niks meer aangedaan aangezien dit veel tijd zou kosten voor een klein puntje (mijn persoonlijke mening).
